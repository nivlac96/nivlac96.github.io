NextGaming Skill-Based Slots

While working at Present Creative, I have had the chance to work extensively on NextGaming’s innovative skill-based slot machines, which combine classic arcade gameplay with chance wagering. Soon to be available at high-profile casinos in the USA, these machines have already received several awards, including Casino Player Magazine’s Best of Gaming Editor’s Choice Award in 2018.

For this ongoing project, I am writing core gameplay mechanics and UI in Actionscript 3, creating design tools using C# and Unity, and working on hardware interface using C++.

Learn more about this project at nextgaming.co.

Flicker

This game was designed by a small team for our capstone project at UC Santa Cruz. The game explores emotional themes, has a stunning surrealist aesthetic, and employs a unique mechanic for shifting physics between the second and third dimension. Creating and designing for this mechanic in Unity meant lots of scripting for the camera, making heavy use of the Cinemachine package and vector math for the perspective tranisiton, as well as creating a specialized character controller.

A fully playable download is yet to be released, however the project’s source code is available on github.

A Place To Be

Made with Unity, this project was lead by artist Katie Green at the Digital Arts and New Media department at UC Santa Cruz. This game explores the violence, beauty and cyclical nature of life by allowing players to control several creatures at different places on the food chain. The game has been displayed as part of an art exhibition featuring three monitors, where each player is at a different stage in the game’s cycle. I helped create a day-and-night system that is responsive to player progress, as well as transitional levels that were ultimately cut from the final game.

View and download the game at itch.io.

Phoenix Equinox

A short, beautiful puzzle platformer developed by a small team at UC Santa Cruz. The player controls a phoenix with the power to resurrect forest creatures, and a unique set of movement skills to navigate a forest environment. Using phaser.io, our team designed and created our own character controller with flying mechanics and implemented a sloped tilemap system to create a satisfying tactile experience.

HyPed and the Video Game Level Corpus

HyPed is an experimental framework for modelling game characters as hybrid automata. Through research into similar projects and hands-on coding with the python-based HyPed engine, I explored the possibilities of integrating this system with the Video Game Level Corpus, a collection of codified levels from classic 2D games, intended to be used as a training corpus for machine learning projects.

These projects were developed in tandem by students and faculty at the Expressive Intelligence Studio at UC Santa Cruz. Both projects are available on github.
