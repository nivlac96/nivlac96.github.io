Previous Projects

Unity: Flicker

Flicker is a surrealist puzzle platformer with an emphasis on narrative and emotional tone. Similar to Fez (2012) and Crush (2007), the player possesses the power to switch from the third to the second dimension in order to "compress" the world.


Check out the game website at flickergame.net

Phaser.io: Phoenix EQV

Phoenix Equinox V is a 2D platformer where players take on the role of a gliding phoenix, created for an academic project in 2016. Working in Phaser.io, I implemented the arcade-slopes plugin and wrote code for scene tranistions and asset management.


Check out the game on itch.io

Unity: Control

Control is an abstract 3D puzzle/exploration game developed in Unity as a final project for the Fundamentals of Game Design class. Taking inspiration from titles like Portal and Antichamber, Control challenges players' expectations of the first-person 3D video game environment through spatial puzzles with unconventional solutions. The game features FPS-style controls, a highly dynamic and reactive audio environment, and basic UI.


For a playable demo, email me at: cwalantu@ucsc.edu

WebGL: 3D modeling program

A simple rendering program that allows the user to create 3D surface-rotation objects, and arrange them in a scene with multiple light sources, diffuse and specular lighting, and multiple types of shading. Created with WebGL, JavaScript, HTML and CSS for Intro to Computer Graphics.

GameMaker Studio: Let There Be Color

A 2D puzzle/exploration game, similar mechanically and aesthetically to the 1980 Atari classic Adventure. Some of the design goals were to inherently suggest the existence of hidden pathways within the world, to maintain an intriguing atmosphere and to encourage player exploration. This project was made for the Fundamentals of Game Design class using GameMaker, and was a collaboration with fellow student Ian Carlton.


For a playable demo, email me at: cwalantu@ucsc.edu
